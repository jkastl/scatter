🎮 How to Play

Goal

Score points by naming something in each of 10 categories. Every answer must start with that round's letter.

Each Round

A letter is rolled and shown to everyone. Then players take turns one at a time in secret — each player fills in answers while the timer runs, then passes the phone. No one sees anyone else's answers until all players have submitted.

Scoring

All answers are reviewed together on the scoring screen. Valid answers are pre-approved automatically — tap any answer to toggle it on or off. The border color shows which player the answer belongs to.

Duplicates

If two players wrote the same answer for the same category, neither scores — both are flagged "Duplicate." If a player accidentally used the same answer for two different categories, only their first use scores; the repeat is flagged.

Wrong Letter

Answers that don't start with the round's letter are flagged automatically and default to zero. You can tap them to override and award the point anyway — handy for nicknames or edge cases.

During a Turn

Use the ⏸ pause button to hide answers and stop the clock — useful if you need a break mid-turn. The ■ stop button ends the game early and returns to the main menu.

Winning

Most points after all rounds wins. Ties are celebrated equally! 🎉
Stop the game?
